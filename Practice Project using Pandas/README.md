# 🐼 Practice with Pandas

## Overview

This repository contains a collection of hands-on exercises completed while learning and practicing data analysis with Python and Pandas. The notebooks focus on developing core data manipulation, cleaning, exploration, and analysis skills using real-world datasets.

## Objectives

* Strengthen proficiency in Pandas.
* Practice data cleaning and preprocessing techniques.
* Perform exploratory data analysis (EDA).
* Work with grouping, aggregation, sorting, and filtering operations.
* Build confidence in analyzing real-world datasets.

## Projects Included

### 1. Customer Call List Cleaning & Analysis

**Dataset:** Customer Call List

Key activities:

* Data cleaning and preprocessing.
* Handling missing values.
* Removing duplicates.
* Standardizing phone number formats.
* Cleaning text fields.
* Performing exploratory analysis on customer data.

**Skills Practiced:**

* Pandas Data Cleaning
* Data Wrangling
* Missing Value Treatment
* Exploratory Data Analysis

### 2. World Population Exploratory Data Analysis

**Dataset:** World Population Dataset

Key activities:

* Exploring dataset structure and statistics.
* Analyzing population trends and growth rates.
* Grouping and aggregating data by continent.
* Performing correlation analysis.
* Creating visualizations to uncover patterns.

**Skills Practiced:**

* Exploratory Data Analysis (EDA)
* Data Aggregation
* Statistical Analysis
* Data Visualization
* Correlation Analysis
