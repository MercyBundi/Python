# 📞 Customer Call List Data Cleaning Project

## Project Overview

In this project, I used Python and Pandas to clean and prepare a customer call list dataset for analysis. The dataset contained common data quality issues such as duplicate records, inconsistent formatting, missing values, and unnecessary characters. The objective was to transform the raw data into a clean and structured dataset suitable for business use.

## Business Problem

Organizations rely on accurate customer contact information for sales, marketing, and customer support activities. Poor-quality data can lead to failed customer outreach, duplicate communications, and inefficient operations. This project demonstrates the process of identifying and resolving data quality issues to improve data reliability.

## Dataset

The dataset contains customer contact information, including names, phone numbers, addresses, and other customer-related details. The data was intentionally messy to simulate real-world data cleaning scenarios.

## Tools and Technologies

* Python
* Pandas
* Jupyter Notebook

## Data Cleaning Tasks

* Removed duplicate customer records.
* Standardized phone number formats.
* Handled missing and null values.
* Removed unnecessary symbols and whitespace.
* Cleaned and formatted text fields.
* Filtered irrelevant records.
* Created a structured and analysis-ready dataset.

## Key Learning Outcomes

Through this project, I gained hands-on experience in addressing common data quality issues encountered in real-world datasets. The project strengthened my understanding of Pandas operations and reinforced the importance of data cleaning as a critical step in the data analysis workflow.
